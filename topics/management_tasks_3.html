        • Functions or Steps

          We consider for function (step) any combination of command, operators and data set that aim to get one specific result. Combination of functions (steps) form business task.

        • Data and DTO Access Levels

          Data and DTO might have different security access level. Data access level may be assigned to data or DTO types to define different methods to manage and secure them. It depends on enterprise policy.

        • DTO and their types

          This section may be useful for front-end developers, business and technical analytics, pre-sales, sales and even marketing to some point. Data Transfer Object (DTO) usually use to simplify calling data and function that are needed to handle specific task and run needed functions
